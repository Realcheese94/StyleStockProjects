<H1>스타일 스톡</H1>

<H3>스타일 스톡은 자신의 옷장을 데이터 베이스 화 시켜주는 어플리케이션입니다.
자신의 옷을 찍어서 저장 및 수정 삭제가 가능하고,이와 관련된 상품을 대형 쇼핑몰에서 추천 및 해당 상품으로 이동이 가능합니다.
</H3>


<H1>기능</H1>
 <H5>
-로그인/로그아웃
-자신의 옷 데이터 업로드/수정/삭제<br>
-자신의 옷 데이터와 같은 타 상품 추천<br>
-회원 정보 저장 및 수정 관리<br>
-서버와의 채팅/공지 받기<br>
-자신의 옷 이미지로 어플리케이션 내 코디 가능<br>
</H5>
